# Matlab Sound

## 1.Read the audio file

1. `audioread`: Read audio file

   `[y,Fs] = audioread(filename)`

   returns sampled data `y` as an array, and a sample rate for that data `Fs`.

2. `sound`: Convert matrix of signal data tosound

   ```matlab
   sound(y)
   sound(y,Fs)
   ```
